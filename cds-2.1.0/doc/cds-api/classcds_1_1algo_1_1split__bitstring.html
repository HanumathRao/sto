Detailed Description

template<typename BitString, typename UInt = size_t>
class cds::algo::split_bitstring< BitString, UInt >

Cuts a bit sequence from fixed-size bit-string.

The splitter can be used as iterator over bit-string. Each call of cut() or safe_cut() cuts the bit count specified and keeps the position inside bit-string for the next call.

The splitter stores a const reference to bit-string, not a copy. The maximum count of bits that can be cut in a single call is sizeof(UInt) * 8

Cuts next nBits from bit-string.

Precondition: nBits <= sizeof(uint_type) * 8

This function does not manage out-of-bound condition. To control that use safe_cut().

inline

Cuts up to nBits from the bit-string.

Analog of cut() but if nBits is more than the rest of bit-string, only the rest is returned. If eos() is true the function returns 0.
